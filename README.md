# What we are building:

→ Track all your subscriptions (Netflix, Spotify etc)
→ See monthly spending
→ Get alerts for upcoming renewals
→ Beautiful Bento Grid dashboard

# Stack:

→ React + TypeScript + Vite (Frontend)
→ Laravel 12 + Sanctum (Backend API)
→ PostgreSQL (Database)
→ date-fns (Date calculations)
→ Tailwind CSS (Styling)
